# TG Robot UI

一个基于 aiogram 框架的 Telegram 机器人，用于管理和购买 EA 策略许可证。

## 功能特点

- 现代化UI的菜单设计
- EA策略号验证和价格查询
- 在线支付功能
- License 管理功能

## 安装

1. 克隆仓库：
```bash
git clone https://github.com/yourusername/tg-robot-ui.git
cd tg-robot-ui
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 配置环境变量：
创建 `.env` 文件并添加以下配置：
```
BOT_TOKEN=your_bot_token
```

4. 运行机器人：
```bash
python main.py
```
